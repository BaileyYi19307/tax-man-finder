# User flows

## Client
1. See homepage
2. Choose "Find an Accountant"
3. Browse Listings
4. Click Profile 
5. Must sign up to contact

## Signing up (Client):
1. User clicks on Sign up
2. Portal pops up - user has to fill in email address and password 
3. Checks to see if user already exists - if yes, says account already exists
4. If no, saves user in data base and redirects them to their dashboard


## Accountant
1. See homepage
2. Choose "Offer Accounting Services"
3. Create Account 
4. Complete Profile
5. Publish service listing
